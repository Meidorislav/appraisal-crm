# ADR-004: Открытая лицензия (MIT / Apache 2.0)

> Русская версия · [English version](../../ADR-004-open-source-license.md)

**Статус:** На рассмотрении  
**Дата:** 2026-06-15

## Контекст

Клиент требует, чтобы исходный код оставался публично доступным после передачи.
Лицензия должна быть достаточно permissive, чтобы клиент мог свободно
использовать, изменять и распространять ПО, и при этом совместима с
open-source зависимостями проекта (экосистема Go, React и др.).

## Решение

Выпустить весь исходный код под лицензией **MIT** или **Apache 2.0**.
Окончательный выбор будет сделан до первого публичного релиза.

- **MIT** — проще, короче, широко известна
- **Apache 2.0** — включает явный патентный грант, лучше подходит для
  коммерческого использования

## Последствия

**Положительные:**
- Клиент имеет полную свободу использовать, изменять и распространять код
- Совместима со всеми основными open-source зависимостями в стеке
- Привлекает потенциальных контрибьюторов из сообщества

**Отрицательные:**
- Любой может форкнуть и использовать код без обратного вклада
- Нет коммерческой защиты для команды разработки
